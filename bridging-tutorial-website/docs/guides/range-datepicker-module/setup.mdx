#### Source files stubs

Finally, let's create stubs for source files. For JS, create following empty files:

- `src/index.ts`
- `src/NativeRangeDatePickerModule.ts`

For iOS implementation, create:

<Tabs groupId="ios_lang">
<TabItem value="swift" label="ObjC++ & Swift">

- `ios/RangeDatePickerModule.h`
- `ios/RangeDatePickerModule.mm`
- `ios/RangeDatePickerModuleImpl.swift`
- `ios/RangeDatePickerPackage-Bridging-Header.h`

</TabItem>
<TabItem value="objc" label="ObjC++ only">

- `ios/RangeDatePickerModule.h`
- `ios/RangeDatePickerModule.mm`
- `ios/RangeDatePickerModuleImpl.h`
- `ios/RangeDatePickerModuleImpl.mm`
- `ios/RangeDatePickerPackage-Bridging-Header.h`
- `ios/FastisWrapper.swift` (needed to wrap Swift datepicker library)

</TabItem>
</Tabs>
