#### `RangeDatePickerPackage-Bridging-Header.h`

To use some ObjC API in Swift it needs to added to the bridging header.
In our case, we need to use `RCTPresentedViewController` function from `React/RCTUtils.h` in our Swift code, so let's add it here:

```objc title="ios/RangeDatePickerPackage-Bridging-Header.h"
#import <React/RCTUtils.h>
```
