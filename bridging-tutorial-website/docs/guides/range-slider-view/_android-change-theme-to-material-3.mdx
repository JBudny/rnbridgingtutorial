#### Change Android theme to `Material3`

To use `MaterialComponents` library, we need to change the Android app's theme to the Material theme.
To do that, let's navigate to our tutorial app and go to `styles.xml` in Android resources directory:

```diff title="android/app/src/main/res/values/styles.xml"
- <style name="AppTheme" parent="Theme.AppCompat.DayNight.NoActionBar">
+ <style name="AppTheme" parent="Theme.Material3.DayNight.NoActionBar">
```
